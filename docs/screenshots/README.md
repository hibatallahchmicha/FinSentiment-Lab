# Dashboard Screenshots

This folder contains screenshots of the FinSentiment Lab dashboard for documentation and portfolio purposes.

## File Naming Convention

Save screenshots with the following names:

- `01-navigation.png` - Navigation sidebar and real-time prices section
- `02-sentiment-timeline.png` - Sentiment + Price Timeline main chart
- `03-live-metrics.png` - Live Market Metrics (Latest Price, Sentiment, Price Change, etc.)
- `04-statistics.png` - Statistics section showing Daily Sentiment Score and aggregate metrics
- `05-granger-causality.png` - Significant Relationships (Granger causality results table)
- `06-leaderboard.png` - Top 3 Performers (model performance leaderboard)

## How to Add Screenshots

### Method 1: Using Print Screen (Auto-save)
1. Open the Streamlit dashboard at `http://localhost:8502`
2. For each view, press **Print Screen** or **Shift+Print Screen**
3. Open Paint or image editor, paste, and save as PNG with the appropriate filename above

### Method 2: Using Streamlit Screenshot Tools
1. Many views have camera icons in the top-right corner
2. Click the 📷 icon to download the plot as PNG
3. Save with the filename above

### Method 3: Using Browser DevTools
1. Open browser DevTools (F12)
2. Right-click on element → "Capture node screenshot"
3. Save as PNG with the filename above

## Screenshot Size Guidelines

**Recommended dimensions**: 1920x1080 px (16:9 aspect ratio)
- Ensures readability on GitHub and portfolio sites
- Consistent with modern dashboard displays
- Renders well at various zoom levels

**Larger screenshots are acceptable** if they capture important details.
